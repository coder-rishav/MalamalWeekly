How Exchange Rates Work

The exchange rate represents how much of the "To Currency" you get for 1 unit of the "From Currency".

Example:
If 1 USD = 83.50 INR
- From: USD
- To: INR
- Rate: 83.50
Bidirectional Conversion

The system automatically calculates reverse conversions. You only need to create one rate for each currency pair.

Active Status

Only active rates are used for conversions. You can temporarily disable a rate without deleting it.

Important: Always verify rates are accurate before activating them for live transactions.
